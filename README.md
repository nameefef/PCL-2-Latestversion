> 💡 如果 PCL2 无法启动，请先安装 .NET Framework 4.6.2

## 🚀 安装步骤

### 1. 安装 .NET 运行环境（可选）

如果 PCL2 打不开，运行：
```
.NET Framework 4.6.2.exe
```

### 2. 解压 PCL2

1. 下载 `PCL 正式版 2.12.2.zip`
2. 右键 → 解压到 `PCL 正式版 2.12.2\`
3. 进入目录，运行 `PCL 启动器.exe`

## 📖 使用说明

1. 首次启动会自动下载 Java
2. 登录 Microsoft/Xbox 账号
3. 下载游戏版本，开始游玩！
