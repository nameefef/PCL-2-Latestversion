## 简介

**PCL2** 是一款开源的 Minecraft（我的世界）启动器，由 [Hex-Dragon](https://github.com/Hex-Dragon) 开发维护。

本仓库用于追踪 PCL2 的最新正式版发布，及时获取更新。

## 下载方式

### 方式一：GitHub Releases（推荐）

1. 访问 [PCL2 Releases 页面](https://github.com/Hex-Dragon/PCL2/releases)
2. 下载最新的 `PCL2-x.x.x.zip` 文件
3. 解压后运行 `PCL2.exe`

### 方式二：国内镜像下载

- [FastGit 镜像](https://download.fastgit.org/Hex-Dragon/PCL2/releases)
- [PCL2 官方下载站](https://pcl2.download/)

## 安装步骤

1. 下载 `PCL2-x.x.x.zip`
2. 解压到任意目录（建议桌面）
3. 运行 `PCL2.exe`
4. 首次启动会自动下载 Java
